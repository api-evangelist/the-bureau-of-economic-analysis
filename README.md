# The Bureau of Economic Analysis

The Bureau of Economic Analysis (BEA) is an agency within the U.S. Department of Commerce that provides economic data to policymakers, businesses, and the general public. The BEA Data API provides programmatic access to all published economic statistics including National Income and Product Accounts (NIPA), regional data by state and metro area, international accounts, and industry-level GDP data.

**Human URL:** [https://www.bea.gov/resources/for-developers](https://www.bea.gov/resources/for-developers)

## Datasets Available

| Dataset | Description |
|---|---|
| NIPA | National Income and Product Accounts (GDP, personal income, etc.) |
| NIUnderlyingDetail | Underlying detail for NIPA tables |
| MNE | Multinational enterprises — U.S. and foreign multinationals |
| FixedAssets | Net stock, depreciation, and investment in fixed assets |
| ITA | International Transactions Accounts |
| IIP | International Investment Position |
| InputOutput | Supply and use / input-output tables |
| IntlServTrade | International services trade |
| IntlServSTA | International services supplied through affiliates |
| GDPbyIndustry | GDP, gross output, and value added by NAICS industry |
| Regional | State, MSA, and county economic data |
| UnderlyingGDPbyIndustry | Underlying GDP by industry detail |
